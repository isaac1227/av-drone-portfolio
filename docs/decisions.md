# Decisions

## Videos -> YouTube

- Use YouTube as the public video platform.
- Keep video hosting external to reduce cost and simplify delivery.
- Link projects to YouTube URLs from the portfolio.

## CMS -> Sanity

- Use Sanity for works, services and categories.
- Allow content editing without touching code.
- Keep the content model flexible for future growth.

## Emails -> Resend

- Use Resend for contact form emails.
- Handle form submissions through a Next.js route handler.
- Keep the email flow simple and reliable.

## Hosting -> Vercel

- Deploy the site on Vercel.
- Use the native Next.js deployment flow.
- Keep preview deployments available for review.
