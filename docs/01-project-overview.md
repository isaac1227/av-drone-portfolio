# Project Overview

## Context

NATT Aerial Studio es un portfolio web para Natalia Mora, creadora audiovisual especializada en fotografía y vídeo aéreo con dron. El proyecto funciona como MVP comercial y como pieza de portfolio técnico: debe enseñar el trabajo, transmitir confianza y convertir visitas en solicitudes de briefing.

## What the site solves

- Reúne servicios, portfolio y contacto en una sola web.
- Presenta trabajos reales con una interfaz visual premium.
- Mantiene el contenido editable sin necesidad de tocar el código.
- Prioriza SEO y tiempos de mantenimiento bajos.

## Current scope

- Home con hero, servicios y trabajos destacados.
- Portfolio navegable por categoría.
- Fichas de trabajos con soporte para imagen, galería y vídeo.
- Página de servicios con enfoque editorial.
- Página de sobre mí con contexto profesional.
- Formulario de contacto conectado a Resend.
- Studio de Sanity para gestión de contenidos.

## What is intentionally out of scope

- Un CMS personalizado.
- Complejidad editorial innecesaria.
- Pricing público fijo.
- Automatizaciones avanzadas que no aporten valor inmediato.
